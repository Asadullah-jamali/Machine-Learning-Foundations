# Machine Learning Foundations: The Perceptron
This project features a ground-up implementation of the Perceptron learning rule, demonstrating how a simple neural network learns to classify patterns (specifically looking at logic gates like AND/OR).
## Key Features
 * *Manual Weight Updates:* Implements the learning rule where weights (w_1, w_2) are adjusted based on the error between predicted and actual outcomes.
 * *Convergence Logic:* Includes an okcount mechanism to ensure the model only stops training once it perfectly classifies all input patterns in a sequence.
 * *Step-by-Step Trace:* The script prints the net input, error status, and updated weights for every iteration, making the "learning" process transparent.
## Technical Skills
 * *Python Logic:* Use of loops, conditional logic, and state tracking to simulate a training epoch.
 * *Neural Network Fundamentals:* Understanding of activation thresholds, learning rates, and weight optimization.
